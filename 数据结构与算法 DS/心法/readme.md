### 最好 最坏 平均 均摊

> 概率论

- 最好情况时间复杂度（best case time complexity）

- 最坏情况时间复杂度（worst case time complexity）

- 平均情况时间复杂度（average case time complexity）

- 均摊时间复杂度（amortized time complexity）

一般均摊时间复杂度就等于最好情况时间复杂度
